# 关于 IDontGetAI

> **Cognitive Glitch (认知故障)**: 当思维试图解析它无法完全理解的系统时产生的视觉与逻辑断裂。

## 👁️ 愿景

**IDontGetAI** 不是一个完美的知识百科，而是一个**正在进行的思考过程**。在这个 AI 飞速发展的时代，承认"我不懂 (I Don't Get It)" 是一种理性的诚实，也是深入探索的起点。

本项目通过赛博朋克（Cyberpunk）与故障艺术（Glitch Art）的视觉风格，隐喻人类认知在面对庞大复杂系统（如通用人工智能、量子物理、复杂经济系统）时的那种**不稳定性**与**破碎感**。

## ⚡ 核心特性

1.  **沉浸式阅读体验**:
    -   全站采用深色模式，配合霓虹配色（Neon Green/Cyan/Magenta）。
    -   定制的 `PdfViewer` 与 `MarkdownViewer` 提供无干扰的阅读环境。

2.  **去中心化内容源**:
    -   网站本身只是一个**"视窗" (Viewer)**。
    -   实际的知识内容（Markdown 笔记、PDF 书籍）可以托管在 GitHub 的任何公共仓库中。
    -   通过 URL 参数动态拉取内容，实现"一次编写，随处渲染"。

3.  **结构化思维导图**:
    -   首页集成 `NeuralTopologyCanvas`，动态可视化知识节点间的连接。
    -   学科分类覆盖 AI、数学、物理、哲学、心理学等 10+ 领域。

## 🎨 视觉系统

-   **字体**: 使用 Monospace 等宽字体（如 JetBrains Mono, Fira Code 风格）强调"终端/代码"属性。
-   **动效**: 打字机效果 (`TypingAnimation`) 与毛刺效果 (`GlitchText`) 模拟老式 CRT 显示器的信号干扰。
-   **布局**: 采用卡片式 (Bento Grid) 与列表视图结合，兼顾信息密度与可读性。
