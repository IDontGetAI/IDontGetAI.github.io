# AI学习合辑主题

这是一个用于GitHub Pages的AI学习课程和笔记合辑主题，专为IDontGetAI.github.io设计。

## 项目介绍

这个主题是一个现代化的静态网站，用于展示AI学习的课程和笔记合辑。它具有以下特点：

- 🎨 现代化的设计风格
- 📱 完全响应式布局，适配各种设备
- ✨ 流畅的动画和交互效果
- 📚 清晰的课程和笔记分类展示
- 🚀 易于部署和自定义

## 网站结构

```
├── index.html          # 主页HTML文件
├── styles.css          # 样式文件
├── script.js           # JavaScript交互文件
└── README.md           # 部署说明文档
```

## 部署步骤

### 1. 准备工作

确保你已经拥有GitHub账号，并且已经创建了名为`IDontGetAI.github.io`的仓库。

### 2. 克隆仓库

将你的GitHub Pages仓库克隆到本地：

```bash
git clone https://github.com/IDontGetAI/IDontGetAI.github.io.git
cd IDontGetAI.github.io
```

### 3. 添加文件

将本主题的所有文件（`index.html`、`styles.css`、`script.js`）复制到你的仓库目录中。

### 4. 提交和推送

```bash
git add .
git commit -m "Add AI learning theme"
git push origin main
```

### 5. 启用GitHub Pages

1. 登录GitHub，进入你的仓库
2. 点击"Settings"选项卡
3. 在左侧导航栏中选择"Pages"
4. 在"Source"部分，选择"main"分支，然后点击"Save"
5. 等待几分钟，你的网站就会部署完成

### 6. 访问网站

部署完成后，你可以通过以下网址访问你的网站：

```
https://IDontGetAI.github.io
```

## 自定义指南

### 修改内容

1. **修改网站标题和描述**：
   - 编辑`index.html`文件中的`<title>`标签
   - 修改英雄区域（hero section）的标题和描述

2. **添加/修改课程**：
   - 在`index.html`文件中找到`courses-grid`部分
   - 复制现有的课程卡片，修改内容即可

3. **添加/修改笔记**：
   - 在`index.html`文件中找到`notes-grid`部分
   - 复制现有的笔记卡片，修改内容即可

4. **修改关于信息**：
   - 在`index.html`文件中找到`about`部分，修改内容即可

### 修改样式

1. **修改颜色主题**：
   - 编辑`styles.css`文件中的颜色变量
   - 主要颜色包括：
     - 主色调：`#4A90E2`（蓝色）
     - 辅助色：`#50E3C2`（青色）
     - 强调色：`#E94E77`（粉色）

2. **修改字体**：
   - 目前使用的是Google Fonts的`Inter`字体
   - 可以在`index.html`文件中修改字体链接

3. **调整布局**：
   - 可以修改`styles.css`文件中的网格布局、间距等

### 添加新功能

1. **添加新页面**：
   - 创建新的HTML文件，例如`course-detail.html`
   - 在导航栏中添加链接

2. **添加图片**：
   - 可以将图片上传到仓库中，然后在HTML中引用
   - 或者使用外部图片链接

3. **添加更多交互效果**：
   - 编辑`script.js`文件，添加新的JavaScript代码
